## Docker Ruby on Rails Simple api

learning docker simple api Get,Post,Put,Delete

### Run docker
 ```bash
 $ docker-compose up
 ```

if you want run rails commands.

examples:
 ```bash
$ docker-compose run --rm app rails routes
$ docker-compose run --rm app rails db:migrate
 ```
